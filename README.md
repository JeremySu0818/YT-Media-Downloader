# YT Media Downloader

一款簡單易用的 YouTube 影片／音訊下載工具，提供直覺式介面、縮圖預覽、佇列系統與批次下載功能。適合一般使用者快速整理並下載所需的影片或音樂。

---

## 主要功能

* 影片下載（自動選擇最佳畫質或手動選擇解析度）
* 音訊下載（支援多種格式）
* 自動載入縮圖
* 下載佇列與批次處理
* Dark 模式介面
* 下載進度條與狀態顯示

---

## 使用方式

1. 將 YouTube 連結貼上欄位
2. 按「分析資訊」取得縮圖與可下載格式
3. 選擇影片或音訊格式
4. 加入佇列後，可同時下載多個項目
5. 按「下載」開始批次處理
